# gRPC 微服务项目

本项目是Go实战入门路线的第三阶段，贴近生产级微服务场景。

## 项目目标

- Product Service（商品服务）
- Order Service（订单服务）
- 使用 gRPC + Protobuf 通信
- Etcd 作为服务发现
- Viper + 环境变量
- Docker Compose 一键启动

## 目录结构

```
├── proto/
├── product-service/
├── order-service/
└── docker-compose.yaml
```

## 学习重点

- gRPC server/client 的结构
- Protobuf 定义规范
- Etcd 的服务注册与发现
- 多服务的本地开发方式
